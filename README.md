This project is created by **Mithilesh Kumar** for beginners who want to learn **API Testing** using simple and easy-to-understand REST APIs.

The project contains:

- Basic CRUD APIs  
- In-memory data storage (No database required)  
- Swagger UI for manual API testing  
- Perfect for Postman learners  
- Clean and beginner-friendly project structure
